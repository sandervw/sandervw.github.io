Prior Work

- CPRE 185: 2D Game Project

For this project, I worked on creating a 2D Game that served as our major project throughout the second half of the semester. This was a two person project, and was thus not expected to be very large or function-filled, but it was required that we get as much of it working as possible. My partner in the project dropped out of the class not long after it started, so I ended up doing the project by myself. The project was built entirely using C++ and a graphics library designed by the instructor for the class. It was not a large library, and mostly consisted of functions for drawing simple shapes, and some functions to load .jpg image files. The class was my first programming programming experience ever, and it might be one of my favorite programming experience. The instructor was very passionate about his job and starting off my programming experience by building a game was tons of fun. The demo and code involved are long lost, but I still Appreciate the knowledge and skills I gained in that class.
